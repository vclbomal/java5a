Edit MOTM content

This page allows you to edit the content of your Mood Of The Month


Customize your MOTM with the following content:
  • An email subject: this will be the subject of the email sent to employees
  • An email template: this content will be inserted into the email to each employee
  • A form template: this content will be displayed on the page where each employee can vote
Template tokens: You can use the following tokens anywhere in each content:
  • ${name}: the user name
  • ${link}: the link to the MOTM for the current user
  • ${currentMonth}: the current month (ex: January)
